# BWA-CBT  

BWA-CBT adalah aplikasi website ujian online berbasis komputer (Computer Based Test) yang dirancang untuk membantu guru dan manajer dalam menilai pengetahuan siswa atau karyawan secara efisien. Aplikasi ini dilengkapi dengan fitur manajemen soal, ujian, dan autentikasi berbasis Spatie untuk pengelolaan peran pengguna.  

## Fitur Utama  
- **Manajemen Soal dan Ujian**: Membuat, mengelola, dan mengatur soal serta jadwal ujian dengan mudah.  
- **Autentikasi dan Pengelolaan Peran**: Sistem login aman dengan pengelolaan peran pengguna, seperti guru atau manajer, menggunakan Spatie.  
- **Evaluasi Hasil Ujian**: Hasil ujian dapat diakses dan dikelola untuk mempermudah penilaian dan pelaporan.  

## Teknologi yang Digunakan  
- **Laravel**
- **Tailwind CSS**
- **MySQL**
